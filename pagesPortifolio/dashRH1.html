Detalhes do Projeto

Este repositório foi criado para colocar na prática os conhecimentos adquiridos na área de análise de Dados, no curso da HastagTreinamentos .

O objetivo do projeto foi criar um sistema utilizando a ferramenta do PowerBi, para ser aplicado no setor de RH de uma empresa, a fim de proporcionar melhores práticas de análises de dados para melhores tomadas de decisões.
